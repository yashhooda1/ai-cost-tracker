Integration

One line to integrate.
Zero code changes.

Point your SDK at Proxima's proxy URL and add your tracker key. That's it.

Python · OpenAI
Python · Anthropic
Node.js

# Before: standard OpenAI SDK
from openai import OpenAI
client = OpenAI(api_key="your-openai-key")

# After: add 2 lines → instant tracking + savings
from openai import OpenAI
client = OpenAI(
  api_key="your-openai-key",
  base_url="https://proxy.proxima.ai/openai/v1", # ← add this
  default_headers={"X-Tracker-Key": "your-company-key"}, # ← and this
)

# Everything else stays identical — responses, streaming, tools
response = client.chat.completions.create(
  model="gpt-4o",
  messages=[{"role": "user", "content": "Hello!"}]
)
